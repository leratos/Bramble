Einträge erstellen

Der Korrektur-Assistent auf jeder Projektseite erstellt neue Journal-Einträge direkt in diesem Projekt.

  • Nachtrag: nutzt den Status notiz und kann per Bezugs-ID Kontext zu einem bestehenden Eintrag ergänzen.
  • Bugfix: nutzt den Status bugfix und kann per korrigierter Eintrags-ID eine corrects-Relation setzen.
  • Aktionslinks an Einträgen füllen die passende Bezugs-ID automatisch vor.

Offene Punkte

Die Kennzahl zeigt alle effektiv offenen Punkte. Die Listen sind bewusst begrenzte Vorschauen: Dashboard zeigt die neuesten 10, Projektseiten zeigen die neuesten 5.

Ein offener Punkt gilt als geschlossen, wenn ein neuerer Abschluss, Bugfix oder Nachtrag ihn eindeutig per Link, Textverweis, Phase, Titel oder Basis-Titel abdeckt.

Suche und Filter

Die globale Suche nutzt SQLite FTS5. Status, Projekt, Zeitraum und Tags schränken Treffer ein; mehrere Tags werden gemeinsam angewendet.

Für einfache Suche reichen normale Wörter. Für exakte Formulierungen können Begriffe in Anführungszeichen gesetzt werden.
